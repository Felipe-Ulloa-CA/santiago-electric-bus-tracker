# Santiago Electric Bus Tracker 

**Real-Time Electric Mobility Platform (Santiago, Chile)**  
Author: Felipe E. Ulloa • Date: 2025-10-09

This repository presents a portfolio-ready **smart mobility system** for **~6,000 electric buses in Santiago de Chile**.  
It includes a synthetic dataset, a **FastAPI** backend stub, a minimal **Leaflet** front-end, and a **Jupyter Notebook** with a baseline **ETA (arrival time) model**.

> Goal: demonstrate end-to-end architecture (IoT → API → ML → Map UI) and delivery approach typical of a **Technical Program Manager / AI** project.
